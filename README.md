This repository documents hands-on IT labs designed to simulate real-world enterprise Active Directory environments.

The goal of this project is to build and demonstrate practical skills relevant to entry-level IT Support and Help Desk roles, including identity management, Group Policy, file permissions, and basic networking services.

## Environment
- Windows Server 2022
- Windows 10 Pro (domain-joined client)
- VirtualBox
- Active Directory Domain Services (AD DS)
- DNS
- DHCP
- Group Policy Management

## Skills Demonstrated
- Active Directory installation and domain configuration
- User, group, and organizational unit management
- Group Policy creation and validation
- NTFS and share permission configuration
- Printer deployment and drive mapping via GPO
- DHCP server installation and scope configuration
- Troubleshootin
